## 🖥️ Running Locally

No dependencies, no build step needed.

```bash
# Clone the repo
git clone https://github.com/JAYESH-SOLMINDE/Portfolio_01.git
cd Portfolio_01

# Serve locally (Python)
python3 -m http.server 8000

# Open in browser
open http://localhost:8000/portf.html
```

## ✏️ Design Highlights

- **Color palette:** `#E8000D` Red · `#0a0a0a` Black · `#f5f0e8` Cream · `#3DFA3D` Green accent
- **Typography:** Anton (display) + Space Grotesk (body) — Google Fonts
- **Animations:** CSS scroll-reveal, animated counters (easing), marquee tickers, hover micro-animations
- **Project filters:** Category tabs (All / AI-ML / Full-Stack / Healthcare / GovTech) with instant JS filtering
- **Grain texture:** SVG noise overlay for tactile depth
- **Responsive:** 3-col → 2-col → 1-col project grid across breakpoints
